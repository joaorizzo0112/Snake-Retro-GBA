Um jogo da cobra clássico com uma experiência visual imersiva de consola retro. Este projeto foi desenvolvido para oferecer uma estética de Game Boy fiel, com layouts dinâmicos que se adaptam perfeitamente a dispositivos Desktop e Mobile.

---

## Funcionalidades Principais

* **Design Adaptativo Inteligente:**
    * **Modo Desktop:** Layout vertical de Game Boy Clássico.
    * **Modo Landscape (Mobile/Touch):** Transforma-se num Game Boy Advance (GBA) horizontal com botões táteis.
* **Três Modos de Jogo:**
    * **Clássico:** A experiência original sem obstáculos.
    * **Portal:** A cobra atravessa as paredes e aparece no lado oposto.
    * **Bónus:** Itens especiais aparecem temporariamente para pontuação extra.
* **Progressão de Níveis:** A velocidade aumenta conforme a tua pontuação sobe.
* **Efeitos Visuais:** Filtro de *scanlines* e animações retro de 8-bits.
---

## Como Jogar

### No Computador:
* **Setas do Teclado:** Controlam a direção da cobra.
* **Enter:** Inicia o jogo / Pausa.
* **M:** Ativar/Desativar som.

### No Telemóvel ou Tablet:
* **D-Pad Virtual:** Usa os botões direcionais no ecrã.
* **Botão A (P):** Pausa o jogo.
* **Botão B (M):** Silencia o som.
* **Start/Select:** Inicia ou reinicia a partida.

## Tecnologias Utilizadas

* **HTML5 Canvas:** Para a renderização gráfica do jogo.
* **Vanilla JavaScript:** Toda a lógica de colisão, movimento e modos de jogo.
* **CSS3 Avançado:** Grid, Flexbox e `@media` para a construção do hardware virtual sem necessidade de imagens externas.
